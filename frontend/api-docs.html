üîê Authentication

üìö Knowledge Base (RAG)

GET /api/knowledge List all knowledge entries AUTH
Response
200 OK
{ "entries": [ { "source": "FAQ", "category": "general", "chunks": 5 } ] }
POST /api/knowledge Add text to knowledge base AUTH
Request Body
Parameter Type Description
content *required string Text content to add to knowledge base
source string Source name (default: "manual")
category string Category: general, product, faq, policy, service
Example Request
{ "content": "Our business hours are 9 AM to 6 PM, Monday to Friday.", "source": "FAQ", "category": "general" }
Response
200 OK
{ "message": "Knowledge added successfully", "chunksAdded": 1 }
DELETE /api/knowledge/:source Delete knowledge by source name AUTH
Path Parameters
Parameter Type Description
source *required string URL-encoded source name to delete
Response
200 OK
{ "message": "Knowledge deleted successfully" }
POST /api/settings/upload Upload file to knowledge base AUTH
Request

Content-Type: multipart/form-data

Field Type Description
file *required file PDF, DOCX, XLSX, or CSV file (max 5MB)
Response
200 OK
{ "message": "File processed and added to knowledge base", "source": "document.pdf", "chunksAdded": 12 }
